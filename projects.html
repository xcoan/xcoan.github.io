Infract IO data aggregation and visualization

A web app written for the 2016 Indianapolis Civic Hackathon, and is still under active development. It is built using Node Express on top of MongoDB. Using the Google Maps API and a GEOJson file recieved via an AJAX call to OpenIndy, the app plots data of crimes onto a map of Indianapolis, allowing for efficient and effective data aggregation and visualization for citizens and lawmakers alike. Infract IO recieved second place in the hackathon. Work on Infract IO was done in collaboration with fellow hacker Aaron Krueger.

Medlog record keeping for medical professionals

Developed in November 2016 for ArchHacks. Medlog is a server system built on Rails that allows Hospitals and other medical institutions to easily track basic information on their patients. The goal of the application is to minimize the amount of time medical professionals spend time doing paperwork, allowing more time to see patients and take care of emergencies. Medlog is currently undergoing development to be submitted to the GE Health Cloud Innovation Challenge.
